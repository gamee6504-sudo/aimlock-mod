# Aimlock Mod for Minecraft 1.21

Smooth aim lock only on the target under your crosshair.  
Toggle with V key (configurable).  
Fabric client-side mod.

Features:
- Locks aim smoothly to player in crosshair (no side snapping).
- Toggleable keybind.
- Works in 1.21 Fabric.

How to use:
1. Install Fabric 1.21
2. Put aimlock.jar in mods folder
3. In-game: Press V to toggle
